Коллекции в Java — управляй данными с умом!

Коллекции — это структуры данных, которые позволяют хранить и обрабатывать группы объектов. В отличие от массивов, коллекции могут динамически изменять свой размер и предлагают больше возможностей для работы с данными.

Что такое коллекции?

Коллекции — это интерфейсы и классы из пакета java.util, предоставляющие гибкие способы хранения и управления данными.

Пример:

Импортируем ArrayList и List:

import java.util.ArrayList;

import java.util.List;

Создаем список:

List items = new ArrayList<>();

Добавляем элементы:

items.add("меч");

items.add("щит");

Выводим результат:

System.out.println(items);

Результат:

[меч, щит]

Основные типы коллекций

  • List — упорядоченная коллекция, допускающая дубликаты (ArrayList, LinkedList).
  • Set — коллекция без дублирующихся элементов (HashSet, TreeSet).
  • Map — хранит пары ключ-значение (HashMap, TreeMap).

Пример List — список с доступом по индексу

Импорт List и Arrays:

import java.util.List;

import java.util.Arrays;

Создаем список заметок:

List notes = Arrays.asList("Купить материалы", "Завершить проект");

Проходим по списку и выводим каждую заметку:

for (String note : notes) {

  System.out.println("Заметка: " + note);

}

Результат:

Заметка: Купить материалы

Заметка: Завершить проект

Пример Set — уникальные элементы

Попробуем добавить повторяющиеся значения:

Импорт HashSet и Set:

import java.util.HashSet;

import java.util.Set;

Создаем набор чисел:

Set numbers = new HashSet<>();

numbers.add(10);

numbers.add(10);

numbers.add(5);

Выводим содержимое:

System.out.println(numbers);

Результат:

[5, 10]

Пример Map — связь ключ-значение

Импорт HashMap и Map:

import java.util.HashMap;

import java.util.Map;

Создаем инвентарь:

Map inventory = new HashMap<>();

inventory.put("меч", 1);

inventory.put("яблоко", 5);

Выводим количество яблок:

System.out.println("У тебя есть " + inventory.get("яблоко") + " яблок");

Результат:

У тебя есть 5 яблок

Stream API и коллекции

Java 24 позволяет использовать потоки для удобной обработки данных:

Импорт List и Collectors:

import java.util.List;

import java.util.stream.Collectors;

Фильтруем список по условию:

List filtered = items.stream()

  .filter(i -> i.startsWith("м"))

  .collect(Collectors.toList());

Выводим результат:

System.out.println(filtered);

Результат:

[меч]

Коллекции — основа для эффективной работы с данными в Java. Они гибкие, мощные и поддерживаются современными инструментами разработки.
