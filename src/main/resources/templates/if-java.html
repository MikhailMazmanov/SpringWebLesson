Условия в Java — как логические редстоны в Minecraft

В Minecraft ты строишь механизмы с помощью редстоунов, датчиков и факелов. В Java у тебя есть нечто похожее — условный оператор if. Он позволяет программе принимать решения.

Что такое if?

if — это ключевое слово, которое запускает выполнение блока кода только если условие истинно (true). Это как нажатие кнопки: если сигнал есть — механизм сработает.


if (hasKey) {
    System.out.println("Дверь открыта!");
}

Пример из мира Minecraft

Представь, что ты пишешь мод для проверки уровня голода:


int hunger = 3;

if (hunger == 0) {
    System.out.println("Вы слишком голодны! Нужно есть.");
} else if (hunger < 5) {
    System.out.println("Стоит перекусить.");
} else {
    System.out.println("Вы сыты и готовы к приключениям!");
}

Логические операции

Ты можешь комбинировать условия:

  • && — И (оба условия должны быть истинными)
  • || — ИЛИ (хотя бы одно условие должно быть истинным)
  • ! — НЕ (меняет результат на противоположный)

if (hasSword && isMonsterNear) {
    System.out.println("Атакуй!");
} else if (!isDay) {
    System.out.println("Нужно спрятаться!");
}

Используя if, ты создаешь "интеллект" для своей программы — она начинает реагировать на происходящее, как настоящий игрок в Minecraft.
