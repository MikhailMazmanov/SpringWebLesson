Массивы в Java — как контейнеры для данных

Массивы позволяют хранить несколько значений одного типа в одной переменной. Это удобный способ организации данных, когда нужно работать с большим количеством однотипных элементов.

Что такое массив?

Массив — это структура данных, которая содержит фиксированное количество элементов одного типа. Доступ к каждому элементу осуществляется по индексу, начиная с нуля.


// Создание массива из 5 целых чисел
int[] numbers = new int[5];

// Заполнение ячеек
numbers[0] = 10;
numbers[1] = 64;
numbers[2] = 32;
numbers[3] = 5;
numbers[4] = 1;

Краткое создание массива

Можно сразу задать значения при создании:


String[] items = {"меч", "щит", "лук", "яблоко"};
System.out.println("Третий предмет: " + items[2]);

Результат:

Третий предмет: лук

Пример работы с массивами

Представь, что ты проверяешь список своих записей:


String[] notes = {"Купить материалы", "Завершить проект", "Изучить Java 24"};

for (String note : notes) {
    System.out.println("Заметка: " + note);
}

Этот код выведет все элементы массива, словно ты просматриваешь список дел.

Двумерные массивы — как таблицы

Если тебе нужно больше порядка, используй двумерные массивы. Представь, что это таблица с рядами и колонками:


int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6}
};

System.out.println("Элемент [1][2]: " + matrix[1][2]);

Результат:

Элемент [1][2]: 6

Массивы — основа для создания сложных структур данных. Используй их, чтобы управлять большим количеством информации в твоих проектах.
