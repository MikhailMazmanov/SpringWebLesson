Методы String в Java — как инструменты в инвентаре

Как в Minecraft у тебя есть разные инструменты для добычи, строительства и выживания, так и в Java у класса String есть мощные методы для работы с текстом.

Что такое String?

String — это не просто текст. Это объект, который содержит в себе множество полезных функций. Представь его как сундук с инструментами!


String message = "Привет, Майнкрафт!";

Популярные методы String

Метод Описание Пример
length() Возвращает длину строки "Java".length(); // 4
charAt(int index) Возвращает символ по указанному индексу "Hello".charAt(1); // 'e'
substring(int start, int end) Возвращает часть строки "Minecraft".substring(0, 4); // "Mine"
toUpperCase() Переводит строку в верхний регистр "hi".toUpperCase(); // "HI"
toLowerCase() Переводит строку в нижний регистр "HELLO".toLowerCase(); // "hello"
replace(char oldChar, char newChar) Заменяет символы в строке "boom".replace('o', 'x'); // "bx xm"
equals(Object anObject) Сравнивает строки на равенство "apple".equals("Apple"); // false
contains(CharSequence s) Проверяет, содержит ли строка другую строку "Minecraft".contains("craft"); // true

Практический пример

Допустим, ты хочешь проверить, содержит ли никнейм игрока слово "Admin":


String username = "SuperAdmin";
if (username.contains("Admin")) {
    System.out.println("Это админ!");
}

Изучая методы String, ты становишься настоящим мастером текстовой магии! Это поможет тебе создавать сложные логики, обрабатывать ввод пользователя и работать с файлами в своих проектах.

Попробуй использовать эти методы в своём первом Java-проекте в IntelliJ IDEA. Она поддерживает все новинки Java 24, чтобы ты мог работать с последними возможностями языка!
