Переменные в Java — как блоки в Minecraft

Как в Minecraft ты строишь мир из блоков, так и в Java ты создаешь программы с помощью переменных — базовых "строительных элементов" кода.

Что такое переменная?

Переменная — это контейнер для хранения данных. У неё есть:

  • Тип — определяет, какие данные может хранить переменная
  • Имя — уникальный идентификатор переменной
  • Значение — то, что хранится внутри переменной

Примеры переменных в Java

Создадим несколько переменных:


int age = 15;
String name = "Steve";
double speed = 4.3;
boolean isCreative = true;

Каждая из этих переменных — отдельный "блок", который ты можешь использовать в программе!

Основные типы переменных

Тип Описание Пример
int Целое число int level = 5;
double Дробное число double health = 20.0;
String Текст (строка) String playerName = "Notch";
boolean Логическое значение: true или false boolean canFly = false;
