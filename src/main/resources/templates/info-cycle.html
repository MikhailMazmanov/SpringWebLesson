Циклы в Java — повторяй до победного!

Циклы позволяют выполнять один и тот же блок кода многократно. Они полезны, когда нужно обработать данные, перебрать элементы массивов или выполнить задачу несколько раз.

Цикл for — точное количество повторений

Используется, если заранее известно, сколько раз должен выполниться код.


for (int i = 0; i < 5; i++) {
    System.out.println("Итерация №" + i);
}

Результат:


Итерация №0
Итерация №1
Итерация №2
Итерация №3
Итерация №4

Цикл while — пока условие истинно

Выполняется до тех пор, пока заданное условие остаётся верным.


int count = 0;
while (count < 3) {
    System.out.println("Счётчик: " + count);
    count++;
}

Результат:


Счётчик: 0
Счётчик: 1
Счётчик: 2

Цикл do-while — сначала действуй, потом проверь

Отличается тем, что тело цикла выполняется хотя бы один раз, даже если условие не выполнено.


int x = 5;
do {
    System.out.println("Значение x: " + x);
    x--;
} while (x > 0);

Результат:


Значение x: 5
Значение x: 4
Значение x: 3
Значение x: 2
Значение x: 1

Цикл for-each — удобный перебор коллекций и массивов

Упрощает работу с массивами и списками, избавляя от необходимости управлять индексами.


String[] inventory = {"меч", "щит", "лук"};
for (String item : inventory) {
    System.out.println("Предмет: " + item);
}

Результат:


Предмет: меч
Предмет: щит
Предмет: лук

Ключевые слова: break и continue

break — выходит из цикла.
continue — пропускает текущую итерацию и переходит к следующей.


for (int i = 0; i < 10; i++) {
    if (i == 5) {
        continue; // Пропустим число 5
    }
    if (i == 8) {
        break; // Остановимся на 8
    }
    System.out.print(i + " ");
}

Результат:

0 1 2 3 4 6 7

Циклы — мощный инструмент программирования. Используй их, чтобы автоматизировать рутинные операции и управлять потоком выполнения программы.
