Методы в Java — упорядочь логику!

Методы — это блоки кода, которые выполняют конкретную задачу и могут быть вызваны из разных частей программы. Они позволяют организовать код, сделать его понятнее и повторно используемым.

Что такое метод?

Метод — это именованная часть программы, которая может принимать входные данные (аргументы), выполнять действия и возвращать результат.


// Пример простого метода
public static int add(int a, int b) {
    return a + b;
}

Вызов метода:


int result = add(3, 5);
System.out.println("Результат: " + result);

Результат:

Результат: 8

Виды методов

  • Статические методы — принадлежат классу, а не объекту.
  • Нестатические методы — требуют создания экземпляра класса для вызова.
  • Методы с возвращаемым значением — используют ключевое слово return.
  • Методы без возврата — объявляются как void.

Пример метода без возврата


public static void greet(String name) {
    System.out.println("Привет, " + name + "!");
}

// Вызов:
greet("Алекс");

Результат:

Привет, Алекс!

Перегрузка методов

Java позволяет создавать несколько методов с одинаковым именем, но разными параметрами.


public static int multiply(int a, int b) {
    return a * b;
}

public static double multiply(double a, double b) {
    return a * b;
}
        

System.out.println(multiply(3, 4));     // Целые числа
System.out.println(multiply(2.5, 3.0)); // Дробные числа

Результат:


12
7.5

Рекурсивные методы

Метод может вызывать сам себя — это называется рекурсией. Например, вычисление факториала:


public static int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n - 1);
}

// Вызов:
System.out.println(factorial(5));

Результат:

120

Методы помогают делить программу на логические части, облегчают чтение кода и упрощают его отладку. Используй их, чтобы сделать свои проекты чище и мощнее.
